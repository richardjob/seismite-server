Built for LLMs

Native Model Context Protocol Integration

Seismite isn't just a locator manager—it's an AI's eyes on the web. Through standard MCP endpoints exposed by the Seismite Core, AI programming agents like Claude or Cursor can request targeted, sanitized DOM contexts of UI elements in real-time.

  • Smart Context Fetching: The Core provides a stateless MCP tool allowing LLMs to fetch localized HTML context for specific locators.

  • Self-Healing Automation: When UIs change, the AI reads the structural context from the Core to immediately formulate updated robust selectors.

  • AI-Ready Datasets: The extension trims noisy classes and attributes natively, delivering highly-readable semantic trees directly to the prompt context.

  • Dynamic Test Execution: Automation frameworks fetch locators directly from the Seismite Core at runtime. Say goodbye to hardcoded selectors—your tests always use the latest healed locators automatically.

How to Use Seismite

From setup to autonomous testing in three straightforward steps.

1

Spin Up the Core

Deploy the Seismite Core locally. It acts as the central hub—managing the database, hosting the dashboard, and providing the MCP and API endpoints for your framework.

2

Map Your Locators

Install the Chrome Extension. Navigate to your target application and point-and-click to save UI resilient locators securely with their surrounding DOM context.

3

Automate & Heal

Configure your test framework to fetch locators dynamically. When a UI breaks, the Core spots the issue, allowing manual fixes or instant AI self-healing.
